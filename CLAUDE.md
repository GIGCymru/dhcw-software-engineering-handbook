1. **Read the main documentation:**

   ```bash

   ```

2. **Start development server:**

   ```bash
   uv run zensical serve
   ```

3. **Make your changes:**
   - Edit markdown files in `/doc` directory
   - Preview at `http://127.0.0.1:8000/`

4. **Commit and push:**
   - Follow git workflow in AGENTS.md
   - Use clear, descriptive commit messages
